# CRC

CRC：粗(Cū)算 人(Rén)工 成(Chéng)本，以「1.5倍上海社平工资」为核定标准，即 1 CRC = 1.5倍上海社平工资 <br/>
抹零：CRC抹零抹到百位（例如：12399.00 -> 12300.00），由CRC计算的价格也统一抹零抹到百位，避免争议

## CRC取值方法

每年大约七八月份出的最新上海社平工资，以此计算最新CRC，在下一年全年适用 <br/>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;例如： <br/>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;2024年最新上海社平工资出来了，由此计算出的最新CRC，2025年全年适用 <br/>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;2025年最新上海社平工资出来了，由此计算出的最新CRC，2026年全年适用 <br/>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;2026年最新上海社平工资出来了，由此计算出的最新CRC，2027年全年适用 <br/>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;以此类推
